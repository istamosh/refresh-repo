# PHP section

PHP Download:
- https://www.php.net/downloads.php > download the .zip file (x86/x64 Thread Safe) and extract.
- Set environment path. Environment variables > system variables table > Choose Path > Edit > New > Browse > Locate php folder from .zip file
- Restart both VSCode and CMD
- Check on cmd: php -v

- PHP Intelephense
- PHP Server (Right click on file choose "PHP Server: Serve project" for debug)
